# PDF Merger

Small Kotlin + Gradle application to merge PDF files. Contains a REST/controller layer and a PDF service implementation.


## Wanna try?
The website is live under https://www.pdf-merger.app


## Prerequisites
- JDK 17+
- Gradle (use the included `./gradlew` wrapper)

## Build & Run
Run: `./gradlew bootJar && docker compose up -d`
